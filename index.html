Everything stays on your machine

A focused tool for attendance admins who want control, speed, and privacy.

Connect over LAN

Reach attendance devices on your network via the standard TCP attendance protocol (default port 4370), with resilient timeouts.

Pull logs & users

Sync attendance records, employee data, and device info in a few clicks. The UI never hangs if a device drops mid-sync.

USB import

Import flash-drive exports (attlog.dat, user.dat) with automatic encoding (UTF-8 / Windows-1256) and delimiter detection.

Local-first storage

An encrypted local SQLite database. Your attendance data is never uploaded to any server.

Bilingual

Full English and Arabic interface with a persisted locale preference — switch anytime from Settings.

Automatic updates

The in-app updater fetches new releases and installs them silently — no manual download after the first install.

About Windows SmartScreen

The installer isn't signed with a paid code-signing certificate yet, so Windows may show "Windows protected your PC." This is expected for independent software and is not a virus warning. Click More info → Run anyway to continue. Your data and device are unaffected.

Simple license activation

CenturisTime shows a Hardware ID unique to your machine on first launch. Activation is verified locally against a signed license file — no online account, no phoning home.

Open an issue with your Hardware ID, receive a .lic file, then load it from Settings → Load License (or just drag-and-drop the file onto the startup screen).

Request a license

How it works

  • First launch → copy your Hardware ID.
  • Open an issue to request a license.
  • Load the returned .lic file (drag-and-drop supported).
  • Done — activation is tied to this machine only.
